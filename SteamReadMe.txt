[h1]Quasimorph Decrease Weapons Degradation[/h1]


Reduces the durability loss of weapons by 50%.  The reduction multiplier can be set in the mod's configuration file.

[h1]Credit[/h1]

This is the Steam Workshop version of Elija's "Decrease Weapons Degradation" mod.

[h1]Configuration[/h1]

The configuration file will be created on the first game run and can be found at [i]%AppData%\..\LocalLow\Magnum Scriptum Ltd\Quasimorph\DecreaseWeaponsDegradation\config.json[/i].
[table]
[tr]
[td]Name
[/td]
[td]Default
[/td]
[td]Description
[/td]
[/tr]
[tr]
[td]WeaponsDegradationMultiplier
[/td]
[td].5
[/td]
[td]The multiplier applied to durability loss.  Ex:  .25 would reduce the durability loss by 1/4
[/td]
[/tr]
[/table]
